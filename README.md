# EXECUTAR BACKEND

## Acessar diretório assistente-clinico-backend

```bash
cd assistente-clinico-backend
```

## Ativar ambiente virtual
```bash
.venv\Scripts\activate
```

## Instalar requisitos
```bash
pip install -r requirements.txt
```

## Iniciar API
```bash
uvicorn main:app --reload
```

# EXECUTAR FRONTEND

## Acessar outro terminal após rodar o backend

## Instalar dependências

```bash
npm install
```

## Rodar projeto

```bash
npm run dev
```
